## 摘要

### 中文摘要

医学图像分割是计算机辅助诊断的关键技术，其中血管、神经纤维等线状结构的分割因结构细长、拓扑复杂而面临巨大挑战。现有深度学习方法在小目标检测和边界保持方面仍有不足，且大规模预训练模型直接应用于医学图像分割效果有限。

本文提出一种基于LSSeg与SAM-LoRA的级联分割架构。该架构采用"粗到细"的两阶段分割策略：首先利用LSSeg网络生成粗分割Mask，然后将其作为Mask Prompt输入SAM-LoRA进行精细化分割。主要创新点包括：（1）设计了LSSeg与SAM-LoRA的级联架构，实现专用分割网络与大规模预训练模型的有机结合；（2）提出了Prompt Bias优化机制，通过调整logits偏置提升分割召回率；（3）设计了端到端联合训练策略和分层学习率方案，支持两个模块的协同优化。在视网膜血管、神经纤维等多个数据集上的实验结果表明，本文方法在Dice、IoU、clDice等指标上优于现有方法，有效提升了线状结构分割的精度和连续性。

**关键词：** 医学图像分割  线状结构  级联架构  SAM-LoRA  Prompt优化

### Abstract

Medical image segmentation is a key technology in computer-aided diagnosis. The segmentation of line-like structures such as blood vessels and nerve fibers faces great challenges due to their slender and topologically complex characteristics. Existing deep learning methods still have limitations in small object detection and boundary preservation, and large-scale pre-trained models have limited effectiveness when directly applied to medical image segmentation.

This paper proposes a cascaded segmentation architecture based on LSSeg and SAM-LoRA. The architecture adopts a "coarse-to-fine" two-stage segmentation strategy: first, LSSeg network is used to generate coarse segmentation masks, which are then used as Mask Prompts for SAM-LoRA to perform fine segmentation. The main contributions include: (1) designing a cascaded architecture of LSSeg and SAM-LoRA, achieving effective combination of specialized segmentation network and large-scale pre-trained model; (2) proposing a Prompt Bias optimization mechanism to improve segmentation recall by adjusting logits bias; (3) designing an end-to-end joint training strategy with layered learning rate scheme to support collaborative optimization of both modules. Experimental results on multiple datasets including retinal vessels and nerve fibers demonstrate that the proposed method outperforms existing methods in Dice, IoU, clDice and other metrics, effectively improving the accuracy and continuity of line-like structure segmentation.

**Keywords:** Medical image segmentation  Line-like structure  Cascaded architecture  SAM-LoRA  Prompt optimization

## 正文内容

### 前  言

医学图像分割是医学影像分析的核心任务之一，在疾病诊断、治疗规划和疗效评估中发挥着重要作用。其中，视网膜血管、神经纤维等线状结构的精确分割对于眼科疾病早期诊断和神经系统研究具有重要临床价值。然而，由于线状结构具有细长、分支复杂、拓扑结构重要等特点，其精确分割仍面临诸多挑战。

近年来，深度学习技术在医学图像分割领域取得了显著进展，U-Net及其变体已成为主流方法。然而，这些方法在处理细长线状结构时，往往难以同时保证分割精度和结构连续性。另一方面，大规模预训练模型如Segment Anything Model (SAM)展现出强大的零样本分割能力，但如何将其有效应用于特定医学图像分割任务仍需深入研究。

本文针对医学图像中线状结构分割的难题，提出一种基于LSSeg与SAM-LoRA的级联分割方法，旨在通过"粗到细"的分割策略，结合专用网络设计与大规模预训练模型的优势，提升线状结构分割的性能。

### 第1章  绪论

#### 1.1  研究背景和意义

**1.1.1 医学图像分割的重要性**

医学图像分割是指将医学影像中的感兴趣区域（如器官、病变、血管等）从背景中分离出来的过程。精确的分割结果是后续定量分析、三维重建和手术规划的基础，直接影响临床诊断的准确性和治疗方案的制定。

**1.1.2 线状结构分割的应用场景**

- **视网膜血管分割**：视网膜血管的形态变化与糖尿病视网膜病变、高血压视网膜病变、青光眼等多种眼科疾病密切相关。通过精确分割视网膜血管，可以定量分析血管迂曲度、血管直径变化等指标，为疾病早期诊断提供客观依据。

- **神经纤维分割**：在神经科学研究中，神经纤维的精确分割对于理解神经网络结构、分析神经退行性疾病具有重要意义。电子显微镜下的神经纤维图像具有极高的分辨率和复杂的拓扑结构。

**1.1.3 线状结构分割的挑战**

- **结构细长**：线状结构在图像中占比小，像素稀疏，易被忽视
- **拓扑复杂**：存在大量分支、交叉，拓扑结构重要
- **边界模糊**：与背景对比度低，边界难以界定
- **连续性要求高**：断裂会严重影响后续分析
- **尺度差异大**：不同位置的线状结构粗细不一

**1.1.4 研究意义**

本研究旨在开发一种高效、准确的线状结构分割方法，具有重要的理论意义和应用价值：
- 理论上，探索专用网络架构与大规模预训练模型结合的新范式
- 应用上，为临床诊断提供可靠的辅助工具

#### 1.2  国内外研究现状

**1.2.1 传统图像分割方法**

- 阈值分割方法
- 区域生长方法
- 边缘检测方法
- 活动轮廓模型

**1.2.2 基于深度学习的分割方法**

- **U-Net及其变体**：U-Net (Ronneberger et al., 2015) 提出的编码器-解码器结构成为医学图像分割的标准架构。后续工作包括Attention U-Net、U-Net++、nnU-Net等。

- **密集连接网络**：DenseNet (Huang et al., 2017) 提出的密集连接思想促进了特征重用，被广泛应用于分割任务。

- **注意力机制**：CBAM、SE-Net等注意力模块被引入分割网络，增强了模型对关键特征的聚焦能力。

- 传统CNN方法在细长结构分割中难以保持连续性

### 第2章  相关理论基础

#### 2.1  图像分割基础理论

**2.1.1 分割任务定义**

图像分割是将图像划分为若干具有特定含义的区域的过程。形式化地，给定图像 I ∈ R^(H×W×C)，分割任务旨在找到映射函数 f: I → S，其中 S ∈ {0, 1, ..., K-1}^(H×W) 为分割标签图，K为类别数。

**2.1.2 分割任务分类**

- 语义分割：对每个像素进行类别标注
- 实例分割：区分同一类别的不同实例
- 全景分割：语义分割与实例分割的结合

**2.1.3 常用评价指标**

**区域指标：**
- Dice系数 (DSC)：衡量预测与真值的重叠程度
- 交并比 (IoU)：预测与真值的交集与并集之比

**边界指标：**
- ODS (Optimal Dataset Scale)：在最佳阈值下计算的数据集级F1分数
- OIS (Optimal Image Scale)：每张图像最佳阈值下的F1平均
- clDice (centroid Dice)：专门评价线状结构连通性的指标

**距离指标：**
- ASSD (Average Symmetric Surface Distance)：平均对称表面距离
- 95HD (95% Hausdorff Distance)：95%分位的Hausdorff距离

#### 2.2  深度学习分割方法

**2.2.1 卷积神经网络基础**

卷积神经网络通过卷积操作提取图像的局部特征。核心组件包括：卷积层、池化层、批归一化层、激活函数。

**2.2.2 U-Net编码器-解码器架构**

U-Net是医学图像分割的开创性工作，其核心思想是：编码器逐步下采样提取高层语义特征，解码器逐步上采样恢复空间分辨率，跳跃连接融合多尺度特征。

**2.2.3 DenseNet密集连接思想**

DenseNet通过密集连接促进特征重用，每层的输入来自前面所有层的输出，缓解梯度消失问题。

**2.2.4 注意力机制**

注意力机制使模型能够聚焦于关键特征，包括通道注意力（SE-Net）、空间注意力（Non-local）、混合注意力（CBAM、Coordinate Attention）。

#### 2.3  视觉Transformer与SAM模型

**2.3.1 Vision Transformer基本原理**

ViT将图像划分为Patch序列，应用Transformer编码器处理，包括Patch Embedding、Position Embedding、Self-Attention等组件。

**2.3.2 SAM模型架构**

Segment Anything Model由Image Encoder、Prompt Encoder、Mask Decoder三个核心组件构成。

**2.3.3 MedSAM医学图像适配**

MedSAM针对医学图像分割对SAM进行适配，使用大规模医学图像数据集进行微调。

#### 2.4  参数高效微调技术

**2.4.1 全量微调与参数高效微调对比**

全量微调需要更新模型的所有参数，计算成本高，易过拟合。参数高效微调只更新少量参数。

**2.4.2 LoRA原理**

LoRA (Low-Rank Adaptation) 假设模型微调过程中的权重变化是低秩的：W' = W + ΔW = W + BA

**优点：** 参数量大幅减少、训练速度快、可移植性好。

### 第3章  相关工作与技术背景

> 本章介绍本文所采用的已有技术，为后续级联架构设计奠定基础。

#### 3.1  LSSeg线状结构分割网络

**3.1.1 LSSeg简介**

LSSeg（Line-like Structure Segmentation Network）是BM-AI-Lab提出的专门针对线状结构分割的网络架构[1]。该网络针对医学图像中血管、神经纤维等细长结构的分割难题进行了专门设计。

> **引用说明**：LSSeg为已有工作，本文将其作为级联架构的粗分割模块使用。

**3.1.2 核心组件介绍**

**(1) RFCAConv感受野坐标注意力卷积**

RFCAConv是LSSeg的核心创新之一，结合了大感受野卷积与坐标注意力机制：
- 通过深度可分离卷积生成大感受野特征
- 利用坐标注意力捕捉位置信息
- 有效提取细长结构特征

**(2) RFCADenseBlock密集块**

结合DenseNet的密集连接思想与RFCAConv：
- 每层输出与之前的特征拼接
- 促进特征重用
- 渐进式特征提取

**(3) GSC门控跳跃连接**

GSC通过门控机制实现自适应特征融合：
- 门控权重由编码器和解码器特征共同决定
- 自适应调整融合比例
- 优于传统的直接相加或拼接方式

**(4) 损失函数**

LSSeg使用边界追踪损失和纹理抑制损失，专门针对线状结构边界优化。

**3.1.3 LSSeg的优势与局限**

**优势：**
- 专为线状结构设计，分割效果好
- RFCAConv有效结合大感受野与精确定位
- GSC门控跳跃连接提升特征融合效果

**局限性：**
- 作为专用网络，泛化能力有限
- 对极细结构仍存在分割困难
- 边界精细化程度有待提升

#### 3.2  SAM与MedSAM模型

**3.2.1 SAM模型架构详解**

Segment Anything Model (SAM) [2] 是Meta发布的分割基础模型：

**Image Encoder：**
- 基于ViT的图像编码器
- 输出图像的密集特征表示
- 参数量约为90M (ViT-B)

**Prompt Encoder：**
- 支持多种提示类型：点、框、文本、Mask
- 将提示编码为特征向量

**Mask Decoder：**
- 轻量级解码器
- 融合图像特征和提示特征
- 输出分割Mask

**3.2.2 MedSAM医学图像适配**

MedSAM [3] 针对医学图像分割对SAM进行适配：
- 使用大规模医学图像数据集微调
- 采用Box Prompt作为主要提示方式
- 在多种医学图像分割任务上取得优异性能

**3.2.3 SAM应用于医学图像分割的挑战**

- 直接应用效果有限，需要针对医学图像特点进行适配
- Prompt设计对分割效果影响大
- 计算资源需求高

#### 3.3  LoRA参数高效微调方法

**3.3.1 LoRA原理**

LoRA [4] 通过低秩分解实现参数高效微调：

```
W' = W + ΔW = W + BA
其中 B ∈ R^(d×r), A ∈ R^(r×k)，秩 r << min(d, k)
```

**3.3.2 LoRA在SAM中的应用**

将LoRA应用于SAM的Image Encoder：
- 仅微调QKV投影层
- 参数量增加约1%
- 保持预训练知识的同时适应下游任务

#### 3.4  现有方法的局限性分析

**3.4.1 单一模型的局限**

- LSSeg：边界精细化程度不足
- SAM：直接应用医学图像效果有限
- 需要结合两者优势

**3.4.2 级联架构的可行性分析**

将LSSeg与SAM-LoRA级联的可行性：
- LSSeg可提供精准的粗分割Mask

### 第4章  LSSeg-SAM-LoRA级联架构设计

> 本章是本文的核心创新章节，详细阐述级联架构的设计思想、实现细节和训练策略。

#### 4.1  级联架构设计思想

**4.1.1 "粗到细"分割策略**

线状结构分割可采用"粗到细"的两阶段策略：
- **粗分割阶段**：快速定位目标区域，生成初始Mask
- **精细分割阶段**：基于粗分割结果，进行精细化边界分割

这种策略的优势：
- 降低单阶段分割的难度
- 粗分割结果可引导精细分割
- 结合专用网络与通用模型的优势

**4.1.2 为何选择LSSeg + SAM-LoRA**

- **LSSeg**：专为线状结构设计，擅长提取细长特征（已有工作）
- **SAM-LoRA**：大规模预训练模型，具有强大的零样本分割能力
- **级联优势**：LSSeg提供精准的Mask Prompt，SAM-LoRA进行边界优化

**4.1.3 本文创新点概述**

| 创新点 | 说明 |
|--------|------|
| 级联架构 | 首次将LSSeg与SAM-LoRA级联组合 |
| Prompt Bias | 提出logits偏置机制提升召回率 |
| 端到端训练 | 设计联合训练策略 |
| 分层学习率 | 对不同模块采用差异化学习率 |

```python
def get_lora_sam(model_type="vit_b", checkpoint_path, target_size=512, 
                 lora_r=4, lora_alpha=4):
    sam_model = sam_model_registry[model_type](checkpoint=checkpoint_path)
    
    # 冻结原始参数
    for param in sam_model.parameters():
        param.requires_grad = False
    
    # 添加LoRA适配器
    lora_config = LoraConfig(
        r=lora_r, lora_alpha=lora_alpha, target_modules=["qkv"],
        lora_dropout=0.1
    )
    sam_model.image_encoder = get_peft_model(sam_model.image_encoder, lora_config)
    
    # 解冻Mask Decoder
    for param in sam_model.mask_decoder.parameters():
        param.requires_grad = True
    
    return sam_model
```

#### 4.4  Prompt工程优化

**4.4.1 Mask Prompt生成策略**

将LSSeg的输出转换为SAM的Mask Prompt：

```python
# LSSeg输出为Logits，需要下采样到SAM Prompt尺寸
mask_prompt = F.interpolate(
    lsseg_logits,  # [B, 1, H, W]
    (self.mask_prompt_size, self.mask_prompt_size),  # 128×128 for 512输入
    mode='bilinear', align_corners=False
)
```

**关键点**：SAM要求传入Logits而非概率，内部会进行Sigmoid处理。

**4.4.2 Prompt Bias优化机制（本文创新）**

为提升召回率，在Logits上添加偏置：

```python
# prompt_bias: 在 logits 上加偏置，等效于降低阈值
# 例如：logits=-1.38 (概率≈0.2) → 加 bias=1.5 → 0.12 (概率≈0.53)
boosted_logits = lsseg_logits + self.prompt_bias
```

**效果分析**：
- prompt_bias > 0：提升召回率
- prompt_bias = 0：原始行为
- prompt_bias < 0：提升精确率

**4.4.3 Box Prompt辅助策略**

除Mask Prompt外，可选使用Box Prompt：

```python
def _generate_boxes_from_mask(self, mask, H, W):
    """从Mask生成Bounding Box"""
    mask_prob = torch.sigmoid(mask)
    y_indices, x_indices = np.where(mask_prob > 0.5)
    x_min, x_max = x_indices.min(), x_indices.max()
    y_min, y_max = y_indices.min(), y_indices.max()
    # 添加随机扰动
    perturb = 5
    x_min = max(0, x_min - random.randint(0, perturb))
    x_max = min(W, x_max + random.randint(0, perturb))
    ...
```
